## What it does

etch341 talks to the **CH341A** USB programmer over libusb and reads,
erases, writes, and verifies SPI NOR flash + I²C EEPROM chips.
Auto-detects the chip by JEDEC ID, ships a database of 70 common
parts, and handles both 3.3V and 1.8V silicon. One binary serves
both the CLI and a GPUI-based desktop GUI — no subcommand opens the
GUI, any subcommand runs in CLI mode.

<CardGrid>
  <Card title="SPI flash" icon="seti:default">
    Read / erase / write / verify SPI NOR up to 32 MB+ on both 3.3V
    and 1.8V chips. 4-byte addressing for parts above 16 MB.
    Hardware-validated against Macronix MX25U4033E on a GTX 1060
    VBIOS.
  </Card>
  <Card title="I²C EEPROM" icon="seti:powershell">
    Scan the bus, read / write / verify 24Cxx EEPROMs from 128 B
    (24C01) to 64 KB (24C512). Handles the bit-stuffing convention
    for the 24C04 / 08 / 16 sub-families automatically.
  </Card>
  <Card title="Security registers" icon="puzzle">
    Read, erase, and program the W25Q / GD25Q one-time-programmable
    security registers (the `0x48` family) — serial numbers, MAC
    addresses, vendor keys. Decode the SFDP parameter table and the
    SR1/2/3 status bits too.
  </Card>
  <Card title="Cross-platform" icon="setting">
    macOS (arm64 + amd64), Windows (amd64 + arm64), Linux (amd64 +
    arm64). Native installers per platform — `.dmg`, `.msi`, `.deb`,
    `.AppImage`, `.rpm`, `.pkg.tar.zst`.
  </Card>
  <Card title="CLI + GUI" icon="laptop">
    The same binary opens a GPUI desktop window when you double-click
    it and runs as a CLI when you give it a subcommand. Hex viewer
    with find / jump / byte selection / copy, a pop-out activity log,
    a selectable accent color, an on-launch update check, and
    persistent prefs.
  </Card>
</CardGrid>
